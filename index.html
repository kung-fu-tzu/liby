liby.js

Welcome! liby.js is a small JavaScript library with a focus on lightness and performance. Also, it is fair modular.

liby.js is by no means a framework, it provides a minimum of abstraction and makes you use the native JavaScript and DOM as far as it's possible. The only exception is an abstraction level for MSIE 6-8. This level partly implements DOM Level 2 Events, Element interface (Element.prototype), XMLHTTPRequest and some more standarts.

Most parts of the library is paranoidly tested with it own async test framework. This framework has no dependencies on liby.js and can power server-side projects (for ex. ngx_http_js_module).

liby.js powers such sites as Inshaker.ru (since 2007, uses 0.1 and 0.3 branches), RetailStars.ru (since 2008, uses 0.2 branch) and Programica.ru (since late 2006, uses 0.1 branch). Yeap, all sites are in Russian. Please use Google Chrome translate functionality ;)
